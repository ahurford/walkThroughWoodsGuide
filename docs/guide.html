For the t.test if we want to evaluate \(\alpha = 0.05\) if our p-value is greater than our \(\alpha\) both means are not statistically different (based on the datasets). Althought, if the p-value is less than \(\alpha\) then both means are statistically different:

t.test(cat1[,3], cat2[,3], alternative = 'two.sided',paired = TRUE)
