With this data we can make a plot to show correlation between the Average cross sectional vessel area and the number of vessels in the following way:

plot(data1$N..mm..2.~data1$A..mm.2., pch = 19, xlab = "Average cross sectional vessel area", ylab = "Number of vessels per unit cross sectional area", col="blue")

And also we can look at the relation between Wood density and Average of absolute value of latitude

plot(data1$AbsLat..Degrees.~data1$D..g.cm.3., pch = 19, xlab = "Wood density", ylab = "Average of absolute value of latitude", col = 'black')

To analyze the relation between variables we can use the cor(x,y,method = "pearson") function. And it will give us a numeric description of the data’s relation (for more information about this topic:)
